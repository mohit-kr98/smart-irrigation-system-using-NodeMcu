# smart-irrigation-system-using-arduino

The Project Comprises Of Two units : 
1. Main Unit 
2. Sub  Unit
# Components Required
1. Main Unit
 -> NodeMcu
 -> DH11 (Temperature and Moisture Sensor)
 -> Rain Sensor
 -> LED
2. Sub Unit 
 -> NodeMcu
 -> Soil Moisture Sensor
 -> DC Motor Pump(5V)+ Relay
